git简单入门

发表于 2015-03-25

网上有很多git的使用，大都比较乱，不太好用
写个入门级的，便于大家学习和使用，以在centOS系统 命令行使用为例

一 准备工作
1.安装git 2.了解RAS加密算法和SSH协议

二.使用步骤
1.在github.com 注册帐号，创建repository, 比如地址github.com/shrimpma/php-lib
2.下载代码到本地 （相当于svn checkout）
git init //创建新仓库
git clone /path/to/repository 检出仓库
如果是远程则
git clone username@host:/path/to/repository
3.添加和提交
git add
git add *
git commit -m “提交代码信息” //改动已经提交到head,但是还没到远端仓库
git push origin master // 可以把master 换成你想要推送的任何分支
三 常用其他操作
1.更新本地repository
git pull

四.常见问题参考解决方法：

Q:解决git clone提示Permission denied publickey 问题

A:https://help.github.com/articles/error-permission-denied-publickey/

登陆github 使用ssh协议，使用rsa加密算法后，在本地生成公钥和私钥后，需要把公钥的内容放到 github—->settings->ssh keys —>add ssh key

微信支付概要说明

发表于 2015-03-25

此文档只是概要说明微信支付，如需细节 移步 http://mch.weixin.qq.com/wiki/doc/api/index.html

一.微信支付方式有四种 a.刷卡支付 b.公众号支付 c扫描支付 d.APP支付
其中刷卡支付和APP支付 PHP无法实现，前者java实现，后者为Andriod和IOS开发者,
公众号支付和扫描支付 为web开发者适用，都是基于网页的支付，以PHP为例

二.公众号支付
使用场景：微信内网页支付,有H5商城(H5即html5)
准备材料：公众号必须是服务号，申请开通微信支付，有H5商城

三 扫描支付
使用场景：一般在PC端，根据微信支付的规则为不同的商品生成不同的二维码，用户扫描后支付

需要材料：申请开通微信支付模式，收到官方邮件 APPID,微信支付商户号，API密钥，Appsecret
支付流程 参照jd.com
